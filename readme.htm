TaskStep 1.1 - Readme

Introduction

Thanks for downloading TaskStep, a PHP and MySQL-based list/task manager. It is ideal for running on your own computer. If you have a website, it can be installed there and used via an internet connection.

Installation

To run TaskStep on your own computer

  1. Download and install an AMP (Apache/MySQL/PHP) server
    • XAMPP is a free, popular, and cross-platform AMP server. It can be found at http://www.apachefriends.org/en/xampp.html
  2. Extract the contents of this zip file to a folder in your AMP server's web documents folder (usually "htdocs")
    • XAMPP's web documents folder is C:\Program Files\xampp\htdocs
  3. Start your AMP server
  4. Create a new MySQL database called "taskstep" (usually done through phpMyAdmin)
    • If you are using XAMPP, you can access your phpMyAdmin control panel at http://localhost/phpmyadmin/
  5. Create the TaskStep configuration file "config.php" based on the sample "config-sample.php" using a text editor (such as Notepad)
    • Change the MySQL information (server, username, and password) appropriately
    • For XAMPP, the values of 'localhost', 'root', and '' (blank), respectively, will work if you accepted the default settings
  6. Go to the TaskStep file "install/install.php" in your web browswer and follow the on-screen instructions
  7. Once the installation is complete, be sure to delete the "install" folder as a security precaution

To run TaskStep on a web server

  1. Upload the contents of this zip file to a subfolder on your web server
  2. Create a new MySQL database called "taskstep"
  3. Create the TaskStep configuration file "config.php" based on the sample "config-sample.php" using a text editor (such as Notepad)
    • Change the MySQL information (server, username, and password) appropriately
  4. Go to the TaskStep file "install/install.php" in your web browser and follow the on-screen instructions
  5. Once the installation is complete, be sure to delete the "install" folder as a security precaution

Updating

  1. Backup any custom styles
  2. Delete all but config.php files and folders in your TaskStep folder on your AMP server or web server
    • Don't worry, your tasks, settings, and other data is kept in the MySQL database and will not be erased
  3. Unzip the contents of this zip file to your TaskStep folder
  4. Go to the file "install/update.php" in your web browser and follow the on-screen instructions
  5. Once the updating process is complete, be sure to delete the "install" folder as a security precaution

Passwords

Starting with v0.4, TaskStep has a passwords system, so you can now host TaskStep on your own site with a reasonable amount of security. The first time you install TaskStep or upgrade to v0.4 or higher, the password is "taskstep" (without the quotes). You must change this as soon as possible. If you are running TaskStep on your own computer, you can disable sessions to remove the requirement of a password. This is not recommended, and should certainly not be done if you are running it on a website. The system is probably not fool-proof, but it is a start.
